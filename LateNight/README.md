* Two lined prompt
   * Line 1 displays "\<username\>@\<hostname\> \<present working directory\> [\<git current branch\>]"
   * Line 2 displays a colored arrow prompt
   * Git branch changes color if current branch is not clean
   * Colored arrow prompt changes color if exit code from the last command != 0
